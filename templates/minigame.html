Guessing Game

Tic-Tac-Toe Game

  • Description
    an interactive and entertaining activity where a player attempts to identify a concealed number within a predefined range. The player makes successive guesses, and the game provides feedback such as "too high" or "too low" until the correct number is identified, creating an engaging experience that promotes logical thinking and numerical intuition
  • Objective
    correctly guess a hidden number within a specified range.

  • Description
    Also called as Xs and Os, is a game for player/s taking turns marking the spaces in a 3x3 grid with X or O.
  • Rules
    The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row is the winner. If all cells are filled, and no player has three in a row, the game is a tie/draw
